How to play

When you click start (on the game page), it launches a 60 second timer, along with a multiplication question. The objective of this game is to try and answer as many questions correctly as possible within the 60 seconds.

Scoring

Each time you get a question right, it is added to a tally. The same thing happens when you answer a question incorrectly. Once the game is finished, to find the final score the program deducts the amount of incorrectly answered questions from the correctly answered questions tally.
